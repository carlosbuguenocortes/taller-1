# Wikipedia en Grafos

Proyecto de Programación Científica.

## Descripción

Este proyecto modela una red de artículos de Wikipedia utilizando grafos dirigidos.

Cada artículo representa un nodo y cada enlace entre artículos representa una arista dirigida.

Se implementaron:

* Carga de datos desde dataset
* Modelado orientado a objetos
* BFS
* DFS
* PageRank
* Métricas de grados
* Generación de reportes

## Estructura

* `src/` → código fuente
* `dataset/` → archivos del dataset
* `results/` → reportes generados

## Ejecución

Entrar a la carpeta `src` y ejecutar:

```bash
python main.py
```
